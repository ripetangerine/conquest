⭐퀘스트형 글쓰기 & 빠른 글소비 게시판 "conquest" 프로젝트
<br />
<br />

> ### 프로젝트 소개 📖
글을 쓰고, **숏폼과 같이 빠르게 읽는 과정**을 통해서 글을 미디어 소비 대체제로 이용하기 위한 프로젝트입니다.
<br />
<br />

> ### 프로젝트 목적 및 완성도
react 문법 및 프론트, 백엔드의 기본 구조, DB를 공부하기 위해 제작된 프로젝트입니다!
<br />
아직 프론트, 백엔드 로직이 미완성이며 11월 16일까지 완성하는 것을 목표로 하고 있습니다.
<br />
몇몇 프론트 화면 및 백엔드가 덜 구현된 상태이며 전범위적인 틀에 해당하는 구현은 금일 기준(11월 5일) 완료 했습니다.
<br />
<br />

> ### 주요 기능 🤩
- 글 퀘스트 : 글에 관련된 퀘스트(게시글 좋아요 누르기, 글 올리기 등)를 진행하고, 경험치를 얻을 수 있습니다.
- 스와이프 게시글 : space 키를 누르면 랜덤한 다른 게시글로 바로 넘어갑니다.
<br />
<br />

> ### 기술 스택 🛠️
Next.js(TypeScript, css module, api Proxy) + Nest.js(TypeScript, TypeORM) + MySQL(DB, workbench)
figma(link : ""), Notion(link : "none")
<br />
<br />

> ### 설치 방법 ⛏️
**Next.js**
<br />
`npx create-next-app@latest` 
<br />
`npm run dev` 
<br /><br />
**Nest.js** 
<br />
`npm install -g @nestjs/cli` 
<br />
`npm g resource [리소스명]` 
<br />
`npm run start:dev` 
<br /><br />
**MySQL** 
<br />
`MySQL Workbench 로컬 DB 사용` 
<br /><br />
**.env file element** 
<br />
nest -> appModule에 정의되어 있습니다.
```
DB_HOST=
DB_PORT=
DB_USERNAME=
DB_PASSWORD=
DB_DATABASE=
SYNCHRONIZE=
DB_LOGGING=
JWT_SECRET_TEST=
JWT_EXPIRES_IN=
```
